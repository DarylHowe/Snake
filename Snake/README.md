Description
============
The traditional phone game 'SNAKE' re-created in JAVA.

Instructions
============ 
Controls:
Up = 'w' key
Left = 'a' key
Down = 's' key
Right = 'd' key

A self driven project and was chosen as a way to learn about game development in JAVA.
